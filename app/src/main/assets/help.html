GX Checker Manual Instruction

We provide two ways to look up the information of the product.

a.Manually input

Enter the unique product code in the text box, and then press the READ button to query. If query successfully, the screen will display the information of this product, otherwise, it will display "NO RESULT FOUND".

b.Camera scan

Press “START SCANNING” button on the bottom. Authorize the app to use the camera. Place your device 10-15cm directly above the product. Zoom or fine-tune the distance between the device and the product to make the camera get the best focus. When the application recognizes the code, it will automatically execute the query command. If query successfully, the screen will display the information of this product, otherwise, it will display "NO RESULT FOUND".
